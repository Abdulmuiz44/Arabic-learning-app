# Content source pack

Edit curriculum data in this folder, then run:

- `npm run content:build` to validate + generate canonical runtime JSON in `src/data/`
- `npm run content:validate` to validate canonical files only

Files in this folder mirror app entity types:

- books.json
- chapters.json
- lesson_sections.json
- flashcards.json
- quizzes.json
- questions.json
- videos.json
- listening_items.json
